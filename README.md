### Проблематика:

Для внедрения голосовых помощников критично важно, чтобы они умели распознавать речь человека и общаться с помощью голоса. Нейросетевые решения подобных задач, как правило, требуют немалых вычислительных мощностей, поэтому их выделяют в отдельные облачные сервисы и предоставляют к ним доступ через Интернет.

**Цель проекта** - реализовать веб-приложение, которое будет демонстрировать возможности внедрённых алгоритмов Speech-to-Text (STT) и Text-to-Speech (TTS).

### Задачи проекта:

1. реализовать веб-страницу с формами для ввода данных в STT, TTS и отображения их результатов. Входные данные в TTS - текст. Входные данные в STT - аудиофайл или речь с микрофона;

2. внедрить в веб-приложение использование алгоритма для распознавания и генерации речи на русском языке. Должна быть возможность масштабирования алгоритмов TTS, STT;

3. реализовать бэкенд, связывающий веб-страницу с алгоритмами STT и TTS;

4. реализовать и внедрить алгоритм, проставляющий знаки пунктуации (точки и запятые) для результата распознавания речи (STT);

**Соответственно, основные части проекта:** сервисы, непосредственно взаимодействующие с алгоритмами STT и TTS; frontend-часть из 1 веб-страницы; backend-часть, связывающая frontend с STT/TTS сервисами.

### Планируемые результаты проекта:

реализованное веб-приложение, которое может распознать русскую речь человека с аудиофайла или микрофона; сгенерировать аудиофайл с искусственной речью человека по тексту на русском языке.
